# Explainable Machine Learning for Hepatotoxicity Biomarker Discovery

This repository contains the code used in the study:

"Explainable Machine Learning Identifies Robust Transcriptomic Biomarkers of Drug-Induced Hepatotoxicity Using External Validation in the DrugMatrix Dataset"

## Data Sources

- Open TG-GATEs
- GEO GSE57815 (DrugMatrix)

## Workflow

1. Differential Expression Analysis
2. Random Forest Feature Selection
3. XGBoost Model Development
4. SHAP Interpretation
5. GO and KEGG Enrichment Analysis
6. External Validation using DrugMatrix

## Requirements

Python 3.11

Main packages:
- pandas
- numpy
- scikit-learn
- xgboost
- shap
- gseapy
- scipy
- matplotlib
- seaborn
